AutohotKey-Scripts
==================

This repository contains the scripts I use for AutoHotKeys.

Evernote Templates
------------------
The first script attempt is to create a template note for Evernote.
The template note is for Green Sheet meetings at work.

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Green Sheet Evernote Template
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

=====
Usage
=====
* Right click on GreenSheetTemplate.ahk and click Run Script
* Press Win+t to create a new Green Sheet Note in evernote

============
Requirements
============
* Evernote
* AutoHotKeys
* Python
* Green Sheet Template note stored in "My Documents\EvernoteTemplates" as GreenSheetTemplate.enex
  
  - Must remove the <created></created> and <updated></updated> tags (plus their contents) or the note will have the templates creation date
